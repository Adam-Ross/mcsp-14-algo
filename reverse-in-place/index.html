Reverse in place

This algo deals with changing an array in place. This means that the function will mutate the input array.

Given an array: [1,2,3,4,5]

The algo should return [5,4,3,2,1]

The code should read like this after algo is implemented

const arr = [1,2,3,4,5]
console.log(arr) => [1,2,3,4,5]
reverseInPlace(arr)
console.log(arr) => [5,4,3,2,1]
